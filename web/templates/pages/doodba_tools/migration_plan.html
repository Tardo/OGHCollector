🛠️ Doodba Migration Plan

This tool helps plan an Odoo version upgrade. Upload the current addons.yaml and, for every Odoo version between "From" and "To", it shows how addons.yaml should look, plus a repos.yaml with the open PR/MR merges needed for modules that aren't upstream-merged yet.
*** Modules with neither a merge nor an open migration PR at a given version are listed as missing so you know what still blocks that jump.
